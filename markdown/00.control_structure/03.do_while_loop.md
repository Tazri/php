## do while loop
***Program : 03.do_while_loop.php***
```php
<?php
// here simple do while loop

$num = 0;

do{
    echo "\$num : $num";
    echo PHP_EOL;
    $num++;
}while($num <= 5);

?>
```

***Output : 03.do_while_loop.php***
```
$num : 0
$num : 1
$num : 2
$num : 3
$num : 4
$num : 5
```
